Get Book By ID:

This is a POST route. Get book by ID

key: {id}
values: 2
Example: 
[
                                        {
                                            "_id": "5b06a3c5fb6fc075763fc906",
                                            "id": 2,
                                            "name": "The Girl On The Train",
                                            "type": "Drama",
                                            "price": 225,

                                                          "regularDiscunt": 3 }
